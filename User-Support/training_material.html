ឯកសារបណ្តុះបណ្តាលអ្នកប្រើប្រាស់

ប្រព័ន្ធគ្រប់គ្រងឥណទាន និងគណនេយ្យ

១. សេចក្តីផ្តើម

សូមស្វាគមន៍មកកាន់វគ្គបណ្តុះបណ្តាលប្រព័ន្ធគ្រប់គ្រងឥណទាន និងគណនេយ្យ។ វគ្គនេះត្រូវបានរៀបចំឡើងដើម្បីបំពាក់នូវចំណេះដឹង និងជំនាញចាំបាច់ដល់លោកអ្នកក្នុងការប្រើប្រាស់ប្រព័ន្ធថ្មីនេះប្រកបដោយប្រសិទ្ធភាព។ យើងនឹងផ្តោតលើការអនុវត្តជាក់ស្តែង និងឧទាហរណ៍ ដើម្បីឱ្យលោកអ្នកយល់ដឹងកាន់តែច្បាស់។

២. គោលបំណងនៃវគ្គបណ្តុះបណ្តាល

នៅចុងបញ្ចប់នៃវគ្គនេះ អ្នកចូលរួមនឹងអាច៖

  • ចូលប្រើប្រាស់ និងរុករកក្នុងប្រព័ន្ធបានយ៉ាងងាយស្រួល។
  • គ្រប់គ្រងព័ត៌មានអតិថិជន។
  • ស្នើសុំ អនុម័ត និងគ្រប់គ្រងប្រាក់កម្ចី។
  • កត់ត្រាការទូទាត់រំលស់ និងដោះស្រាយការពិន័យ។
  • កត់ត្រាប្រតិបត្តិការចំណូល និងចំណាយគណនេយ្យ។
  • បង្កើត និងមើលរបាយការណ៍សំខាន់ៗ។
  • ដោះស្រាយបញ្ហាទូទៅ។

៣. ម៉ូឌុលបណ្តុះបណ្តាល

ម៉ូឌុលទី ១: ទិដ្ឋភាពទូទៅ និងការចូលប្រើប្រាស់ប្រព័ន្ធ (60 នាទី)

  1. សេចក្តីផ្តើមអំពីប្រព័ន្ធ និងគោលបំណង (១៥ នាទី)
  2. របៀបចូល Login និង Logout (១០ នាទី)
  3. ការរុករកក្នុង Dashboard និង Menu (១៥ នាទី)
  4. សំណួរ និងចម្លើយ (២០ នាទី)

សកម្មភាពអនុវត្ត:

  • អនុវត្តការ Login និង Logout។
  • ស្វែងរក Menu ផ្សេងៗនៅក្នុងប្រព័ន្ធ។

ម៉ូឌុលទី ២: ការគ្រប់គ្រងអតិថិជន (Client Management) (90 នាទី)

  1. របៀបបង្កើតអតិថិជនថ្មី (៣០ នាទី)
  2. របៀបកែប្រែ និងលុបព័ត៌មានអតិថិជន (២០ នាទី)
  3. របៀបមើលប្រវត្តិអតិថិជន (២០ នាទី)
  4. សំណួរ និងចម្លើយ (២០ នាទី)

សកម្មភាពអនុវត្ត:

  • បង្កើតអតិថិជនសាកល្បងថ្មីចំនួន ២ នាក់។
  • កែប្រែព័ត៌មានអតិថិជនដែលបានបង្កើត។

ម៉ូឌុលទី ៣: ការគ្រប់គ្រងប្រាក់កម្ចី (Loan Management) (180 នាទី)

  1. ការស្នើសុំកម្ចីថ្មី និងការគណនាកាលវិភាគរំលស់ (៤៥ នាទី)
  2. ការអនុម័ត/បដិសេធកម្ចី (៤៥ នាទី)
  3. ការកត់ត្រាការទូទាត់រំលស់ និងការពិន័យ (៤៥ នាទី)
  4. ការបិទកម្ចី (១៥ នាទី)
  5. សំណួរ និងចម្លើយ (៣០ នាទី)

សកម្មភាពអនុវត្ត:

  • បង្កើតកម្ចីសាកល្បងចំនួន ៣ សម្រាប់អតិថិជនដែលបានបង្កើត។
  • អនុវត្តការទូទាត់រំលស់ចំនួន ២ ដងសម្រាប់កម្ចីនីមួយៗ (មួយដងទាន់ពេល មួយដងយឺតយ៉ាវ)។

ម៉ូឌុលទី ៤: ការគ្រប់គ្រងគណនេយ្យ (Accounting Management) (120 នាទី)

  1. ការកត់ត្រាប្រតិបត្តិការចំណូល/ចំណាយ (៤៥ នាទី)
  2. ការមើលរបាយការណ៍ចំណេញ-ខាត (៣០ នាទី)
  3. ការមើលរបាយការណ៍តុល្យការ និងលំហូរសាច់ប្រាក់ (៣០ នាទី)
  4. សំណួរ និងចម្លើយ (១៥ នាទី)

សកម្មភាពអនុវត្ត:

  • កត់ត្រាចំណូល (ឧទាហរណ៍៖ ប្រាក់ចំណូលពីការប្រាក់) ចំនួន ៣ ប្រតិបត្តិការ។
  • កត់ត្រាចំណាយ (ឧទាហរណ៍៖ ថ្លៃការិយាល័យ ថ្លៃពលកម្ម) ចំនួន ៣ ប្រតិបត្តិការ។
  • បង្កើតរបាយការណ៍ចំណេញ-ខាតសម្រាប់រយៈពេលជាក់លាក់មួយ។

ម៉ូឌុលទី ៥: ការគ្រប់គ្រងអ្នកប្រើប្រាស់ និងសន្តិសុខ (សម្រាប់ Admin) (60 នាទី)

  1. ការបង្កើត/កែប្រែ/លុបគណនីអ្នកប្រើប្រាស់ (២០ នាទី)
  2. ការកំណត់តួនាទី និងសិទ្ធិចូលប្រើប្រាស់ (២០ នាទី)
  3. ការពិនិត្យ User Activity Log (១០ នាទី)
  4. សំណួរ និងចម្លើយ (១០ នាទី)

សកម្មភាពអនុវត្ត:

  • បង្កើតគណនីអ្នកប្រើប្រាស់ថ្មីចំនួន ២ (មួយសម្រាប់ Loan Officer, មួយសម្រាប់ Accountant)។
  • សាកល្បង Login ដោយប្រើគណនីថ្មី។

៥. សេចក្តីសន្និដ្ឋាន

ខ្ញុំ/យើងជឿជាក់ថាវគ្គបណ្តុះបណ្តាលនេះ បានផ្ដល់នូវចំណេះដឹង និងជំនាញចាំបាច់ដល់លោកអ្នកក្នុងការប្រើប្រាស់ប្រព័ន្ធថ្មីនេះ។ សូមកុំស្ទាក់ស្ទើរក្នុងការសួរសំណួរ ឬទំនាក់ទំនងក្រុមគាំទ្រប្រសិនបើអ្នកជួបបញ្ហា។
