ឯកសារណែនាំអ្នកប្រើប្រាស់

ប្រព័ន្ធគ្រប់គ្រងឥណទាន និងគណនេយ្យ

១. សេចក្តីផ្តើម

ឯកសារណែនាំអ្នកប្រើប្រាស់នេះ ត្រូវបានបង្កើតឡើងដើម្បីជួយបុគ្គលិករបស់ [ឈ្មោះក្រុមហ៊ុនអតិថិជន] ឱ្យប្រើប្រាស់ប្រព័ន្ធគ្រប់គ្រងឥណទាន និងគណនេយ្យ (Loan Installment & Accounting System) បានយ៉ាងមានប្រសិទ្ធភាព។ វានឹងណែនាំអ្នកពីរបៀបចូលប្រើប្រាស់ ប្រើប្រាស់មុខងារសំខាន់ៗ និងបង្កើតរបាយការណ៍។

២. ការចូលប្រើប្រាស់ប្រព័ន្ធ (System Access)

ដើម្បីចូលប្រើប្រាស់ប្រព័ន្ធ សូមអនុវត្តតាមជំហានខាងក្រោម:

  1. បើក Web Browser (Google Chrome, Mozilla Firefox, Microsoft Edge)។
  2. វាយបញ្ចូលអាសយដ្ឋាន URL របស់ប្រព័ន្ធ: [អាសយដ្ឋាន URL របស់ប្រព័ន្ធ]។
  3. អ្នកនឹងឃើញទំព័រ Login ។ បញ្ចូល ឈ្មោះអ្នកប្រើប្រាស់ (Username) និង ពាក្យសម្ងាត់ (Password) របស់អ្នក។
  4. ចុចប៊ូតុង "Login" ឬ "ចូល"។
  5. ប្រសិនបើ Login បានជោគជ័យ អ្នកនឹងទៅកាន់ទំព័រ Dashboard របស់ប្រព័ន្ធ។

៣. ទំព័រ Dashboard (ទំព័រដើម)

បន្ទាប់ពី Login បានជោគជ័យ អ្នកនឹងឃើញទំព័រ Dashboard ដែលបង្ហាញពីទិដ្ឋភាពទូទៅនៃប្រតិបត្តិការសំខាន់ៗ ដូចជា:

  • ចំនួនកម្ចីសកម្ម។
  • ចំនួនកម្ចីដែលហួសកាលកំណត់ (Overdue Loans)។
  • ចំនួនទឹកប្រាក់ដែលត្រូវប្រមូល (Pending Collections)។
  • តារាងសង្ខេបចំណូល/ចំណាយប្រចាំខែ។

៤. មុខងារសំខាន់ៗរបស់ប្រព័ន្ធ

៤.១. ការគ្រប់គ្រងអតិថិជន (Client Management)

ដើម្បីបង្កើត កែប្រែ ឬមើលព័ត៌មានអតិថិជន សូមអនុវត្តតាមជំហាន:

  1. ចូលទៅកាន់ Menu "Clients" ឬ "អតិថិជន"។
  2. ដើម្បីបង្កើតអតិថិជនថ្មី: ចុចប៊ូតុង "Add New Client" បញ្ចូលព័ត៌មាន (ឈ្មោះ លេខអត្តសញ្ញាណប័ណ្ណ អាសយដ្ឋាន លេខទូរស័ព្ទ) រួចចុច "Save"។
  3. ដើម្បីកែប្រែព័ត៌មានអតិថិជន: ស្វែងរកអតិថិជនក្នុងបញ្ជី ចុចលើឈ្មោះ/លេខសម្គាល់ រួចចុចប៊ូតុង "Edit" កែប្រែព័ត៌មាន រួចចុច "Save"។
  4. ដើម្បីមើលព័ត៌មានអតិថិជន: ចុចលើឈ្មោះ/លេខសម្គាល់អតិថិជន។

៤.២. ការគ្រប់គ្រងប្រាក់កម្ចី (Loan Management)

ផ្នែកនេះអនុញ្ញាតឱ្យអ្នកគ្រប់គ្រងប្រតិបត្តិការកម្ចីទាំងមូល:

  1. ចូលទៅកាន់ Menu "Loans" ឬ "កម្ចី"។
  2. ដើម្បីស្នើសុំកម្ចីថ្មី:
    1. ចុចប៊ូតុង "New Loan Application"។
    2. ជ្រើសរើសអតិថិជន ឬបង្កើតអតិថិជនថ្មី។
    3. បញ្ចូលព័ត៌មានកម្ចី (ចំនួនប្រាក់កម្ចី អត្រាការប្រាក់ រយៈពេល)។
    4. ប្រព័ន្ធនឹងគណនាកាលវិភាគទូទាត់រំលស់ដោយស្វ័យប្រវត្តិ។
    5. ពិនិត្យព័ត៌មាន រួចចុច "Submit" ឬ "រក្សាទុក"។
  3. ដើម្បីអនុម័ត/បដិសេធកម្ចី: (សម្រាប់ Admin/Loan Officer)
    1. ចូលទៅកាន់បញ្ជីកម្ចីដែលមានស្ថានភាព "Pending"។
    2. ពិនិត្យព័ត៌មានកម្ចី រួចចុច "Approve" ឬ "Reject"។
  4. ដើម្បីកត់ត្រាការទូទាត់រំលស់:
    1. ចូលទៅកាន់បញ្ជីកម្ចី ហើយចុចលើកម្ចីដែលចង់កត់ត្រា។
    2. ចូលទៅកាន់ផ្ទាំង "Installments" ឬ "ការរំលស់"។
    3. ជ្រើសរើសការរំលស់ដែលត្រូវទូទាត់ រួចចុចប៊ូតុង "Record Payment"។
    4. បញ្ចូលចំនួនទឹកប្រាក់ដែលទទួលបាន និងកាលបរិច្ឆេទទូទាត់។
    5. ប្រព័ន្ធនឹងគណនាការពិន័យយឺតយ៉ាវដោយស្វ័យប្រវត្តិ (ប្រសិនបើមាន)។
    6. ចុច "Confirm Payment" ឬ "បញ្ជាក់ការទូទាត់"។

៤.៣. ការគ្រប់គ្រងគណនេយ្យ (Accounting Management)

ផ្នែកនេះអនុញ្ញាតឱ្យអ្នកកត់ត្រាចំណូល ចំណាយ និងមើលរបាយការណ៍ហិរញ្ញវត្ថុ:

  1. ចូលទៅកាន់ Menu "Accounting" ឬ "គណនេយ្យ"។
  2. ដើម្បីកត់ត្រាចំណូល/ចំណាយ:
    1. ចុចប៊ូតុង "New Transaction"។
    2. ជ្រើសរើសប្រភេទ (Income/Expense)។
    3. ជ្រើសរើសគណនី (ឧទាហរណ៍៖ Revenue - Interest, Expense - Salary)។
    4. បញ្ចូលចំនួនទឹកប្រាក់ និងការពិពណ៌នា។
    5. ចុច "Save Transaction"។
  3. របាយការណ៍ហិរញ្ញវត្ថុ:
    1. ចូលទៅកាន់ផ្ទាំង "Reports" ក្នុងផ្នែក Accounting។
    2. ជ្រើសរើសប្រភេទរបាយការណ៍ដែលអ្នកចង់មើល (ឧទាហរណ៍៖ Income Statement, Balance Sheet, Cash Flow Statement)។
    3. កំណត់រយៈពេល (ឧទាហរណ៍៖ ប្រចាំខែ ប្រចាំត្រីមាស ប្រចាំឆ្នាំ)។
    4. ចុច "Generate Report" ដើម្បីមើលរបាយការណ៍។

៥. ការគ្រប់គ្រងអ្នកប្រើប្រាស់ (User Management - សម្រាប់ Admin)

សម្រាប់អ្នកគ្រប់គ្រងប្រព័ន្ធ អ្នកអាចគ្រប់គ្រងគណនីអ្នកប្រើប្រាស់ផ្សេងៗ:

  1. ចូលទៅកាន់ Menu "Admin" > "User Management"។
  2. អ្នកអាច "Add New User" កែប្រែព័ត៌មាន ឬ "Deactivate" គណនីអ្នកប្រើប្រាស់។
  3. កំណត់ "Role" សម្រាប់អ្នកប្រើប្រាស់នីមួយៗ (Admin, Loan Officer, Accountant) ដើម្បីគ្រប់គ្រងសិទ្ធិចូលប្រើប្រាស់។

៦. ការដោះស្រាយបញ្ហាទូទៅ (Common Troubleshooting)

  • មិនអាច Login បាន: ពិនិត្យ Username និង Password របស់អ្នក។ ត្រូវប្រាកដថា Caps Lock មិនបានបើក។ ប្រសិនបើនៅតែមិនបាន សូមទាក់ទងអ្នកគ្រប់គ្រងប្រព័ន្ធ។
  • ប្រព័ន្ធដើរយឺត: ពិនិត្យមើលការភ្ជាប់អ៊ីនធឺណិតរបស់អ្នក។ ព្យាយាម Refresh ទំព័រ ឬបិទ Browser រួចបើកឡើងវិញ។
  • ទិន្នន័យមិនបង្ហាញត្រឹមត្រូវ: ត្រូវប្រាកដថាអ្នកបានបញ្ចូលទិន្នន័យបានត្រឹមត្រូវ។ ប្រសិនបើបញ្ហានៅតែបន្ត សូមទាក់ទងផ្នែកគាំទ្របច្ចេកទេស។

៧. ការគាំទ្របច្ចេកទេស (Technical Support)

ប្រសិនបើអ្នកជួបប្រទះបញ្ហាដែលមិនអាចដោះស្រាយបាន សូមទំនាក់ទំនងទៅកាន់ក្រុមគាំទ្របច្ចេកទេស:

  • លេខទូរស័ព្ទ: [លេខទូរស័ព្ទគាំទ្រ]
  • អ៊ីមែល: [អ៊ីមែលគាំទ្រ]
  • ម៉ោងធ្វើការ: [ឧទាហរណ៍៖ ៨:០០ ព្រឹក - ៥:០០ ល្ងាច, ច័ន្ទ - សុក្រ]

សូមអរគុណសម្រាប់ការប្រើប្រាស់ប្រព័ន្ធគ្រប់គ្រងឥណទាន និងគណនេយ្យរបស់យើងខ្ញុំ។
